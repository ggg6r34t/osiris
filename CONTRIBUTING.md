# Contributing to Osiris

Thanks for your interest in contributing! Contributions, issues, and feature requests are welcome.

## Ways to Contribute

- Report bugs
- Suggest features
- Improve documentation
- Submit code changes

## Getting Started

1. Fork the repository and create your branch from `main`.
2. Create a virtual environment and install dependencies:
   - Windows PowerShell:
     - `python -m venv venv`
     - `./venv/Scripts/Activate.ps1`
   - macOS/Linux:
     - `python3 -m venv venv`
     - `source venv/bin/activate`
   - Install dependencies:
     - `pip install -r requirements.txt`

## Development Guidelines

- Keep changes focused and minimal.
- Follow the existing code style and project structure.
- Add or update documentation when behavior changes.
- Ensure CLI usage remains clear and accurate.

## Testing

If you add functionality, include basic validation steps in your PR description (what you ran and what you verified).

## Submitting Changes

1. Commit with clear, descriptive messages.
2. Open a pull request with:
   - Summary of changes
   - Motivation/context
   - Any relevant screenshots/logs

## Code of Conduct

Be respectful and constructive in discussions. Harassment and abuse are not tolerated.

## Security Issues

Please report security vulnerabilities via the process in [SECURITY.md](SECURITY.md).
